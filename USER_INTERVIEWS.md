# User Interviews

These were feedback interviews with technically aware college peers and early AI-tool users. They are not startup buyers, but they helped validate whether the product story, audit flow, and dashboard were understandable.

## Interview 1

**Name / initials:** Subodh

**Role and company stage:** College student, early builder / AI tool user

**Context:** I showed him the Spendlens landing page and audit dashboard and asked how understandable and useful it felt for someone comparing AI tool costs.

**Direct quotes:**

- "The dashboard looks professional, but I first wanted to know what data I need to enter."
- "The savings cards make the value clear quickly."
- "The Credex button makes sense after seeing the savings number."

**Most surprising thing they said:** He said the dashboard looked credible, but the audit form needed to make the required input feel obvious before asking for details.

**What it changed about the design:** It reinforced the need for clear step labels, dynamic summary cards, and a demo report path so users understand the product before committing time.
