# Spendlens

A modern B2B SaaS frontend for AI infrastructure spend audits and optimization.

## Frontend

The React + Vite frontend lives in `frontend/`.

```bash
cd frontend
npm install
npm run dev
```

Open `http://127.0.0.1:5173/` for the landing page or `http://127.0.0.1:5173/app` for the audit results dashboard.
